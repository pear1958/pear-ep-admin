## 安装使用

- 获取项目代码

```
git clone git@github.com:v-geek/EP-Admin.git
```

- 安装依赖

```
cd Ep-Admin
pnpm install
```

- 运行

```
新建 .env.development 文件
cp .env.development.default -> .env.development
pnpm run dev
```

- 打包

```
pnpm run build:pro
```

#### 注意事项

- 使用keep-alive记得 页面name(默认为文件名) 与route.name配置的一样
